# Iterações

Registro de todas as iterações do projeto Crianex, seguindo a metodologia **FDD (Feature Driven Development)** com gestão de fluxo **Scrumban Enxuto**.

Cada iteração entrega um conjunto coeso de **Características de Produto (CPs)** orientadas a valor de negócio, e inclui: planejamento por funcionalidade, cerimônias de refinamento, gravações e atas de reunião.

## Processo FDD por Iteração

Cada iteração percorre as etapas 3–5 do FDD:

```
Iteration Planning (Planejar por funcionalidade)
        ↓
Refinamento de features (Projetar por funcionalidade)
        ↓
Desenvolvimento + inspeções formais (Construir por funcionalidade)
        ↓
Validação com cliente → Entrega via PR → Retrospectiva
```
